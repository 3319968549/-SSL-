## 使用流程

### 1. 启动系统

按照以下顺序启动：

```bash
# 终端1: 启动CA
python CA.py
# 终端2: 启动服务器
python server.py
# 终端3: 启动客户端（终端版）
python client.py
```

### 2. 登录/注册

启动客户端后，会提示选择：

```
请选择: [1] 登录  [2] 注册  [q] 退出:
```

- **选择 1**: 登录已有账号
- **选择 2**: 注册新账号
- **选择 q**: 退出程序

#### 注册新用户

- 用户名：不超过8个字符
- 密码要求：至少8个字符

#### 登录

- 输入用户名和密码
- 连续3次失败会被锁定60秒

### 3. 进入聊天室

登录成功后，会自动进行SSL握手，然后进入聊天室。

## 命令说明

### 基本命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/help` | 显示帮助信息 | `/help` |
| `/users` | 显示在线用户列表 | `/users` |
| `/msg <用户> <消息>` | 发送私聊消息 | `/msg alice 你好` |
| `/file <用户> <路径>` | 发送文件 | `/file alice /path/to/file.txt` |
| `/quit` 或 `/exit` | 退出聊天室 | `/quit` |

### 发送消息

#### 群聊消息

直接输入消息内容，按回车发送：

```
你好，大家好！
```

#### 私聊消息

使用 `/msg` 命令：

```
/msg alice 你好，这是私聊消息
```

#### 发送文件

使用 `/file` 命令：

```
/file alice /home/user/document.txt
```

支持的文件类型：
- 文本文件：`.py`, `.doc`, `.txt`, `.docx`
- 图片文件：`.jpg`, `.png`
- 视频文件：`.avi`, `.mp4`

## 消息显示格式

- **群聊消息**：`[时间] 用户名: 消息内容`（蓝色）
- **私聊消息（发送）**：`[时间] 你 -> 用户名: 消息内容`（紫色）
- **私聊消息（接收）**：`[时间] 用户名 -> 你 (私聊): 消息内容`（紫色）
- **系统消息**：`[!] 消息内容`（黄色）
- **错误消息**：`[-] 错误信息`（红色）
- **成功消息**：`[+] 成功信息`（绿色）

## 文件传输流程

1. **发送方**：使用 `/file` 命令发送文件请求
2. **接收方**：收到文件请求提示，输入 `y` 接收或 `n` 拒绝
3. **传输**：文件分片传输，显示进度
4. **完成**：传输完成后显示确认消息

## 配置服务器地址

如果需要连接到不同的服务器，编辑 `client.py` 文件：

```python
server = ("服务器IP地址", 9999)  # 例如: ("192.168.1.100", 9999)
```

## 注意事项

1. **必须先启动CA和服务器**，再启动客户端
2. **SSL握手**：首次登录会自动进行SSL握手，可能需要几秒钟
3. **文件路径**：发送文件时使用绝对路径或相对路径都可以
4. **消息加密**：所有消息都经过AES加密传输
5. **在线用户列表**：使用 `/users` 命令查看当前在线用户
